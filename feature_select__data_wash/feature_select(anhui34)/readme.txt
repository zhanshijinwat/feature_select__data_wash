文件介绍:
1.20210228_raw_anhui_data――原始的安徽气象和收资
	info――各场站原始收资数据
	anhui_wind_W00{}――不同气象源的气象数据
2.processed_data
	W00{}_processedv{}――经过特征筛选后的气象数据(需要运行脚本之后才会生成)
3.jsoin
	v{}
		W00{}.json――不同气象源筛选出来的特征及其最大最小值
	feature_config.json――所有气象源筛选出来的特征，用于最后模型训练
4.特征筛选.ipynb――利用特征筛选算法计算不同气象源中各个气象特征的分值
5.安徽风电_W00{}.ipynb――根据原始气象文件和筛选出来的特征生成算法平台需要的数据格式，
		            具体的代码注释在安徽风电_W001.ipynb中

使用方法：首先运行特征筛选.ipynb文件，运用特征筛选算法，筛选出分值较高的特征，
然后打开安徽风电_W001{}.ipynb，将usecols中的内容手动改成筛选出来的特征，该脚本
将会生成最终符合要求的数据格式，以及不同气象的json文件。
